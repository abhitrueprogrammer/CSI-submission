# Python Grep Implementation
usage:
```bash
python3 grep [options] pattern filename
```
Working options include as of now:

  * caseInsensitive: '-i'
  
  * Show only line count of matching : '-c'
  
  * invert: '-v'
  
  * wordOnly: '-w'
    
  * show Line count: '-n'

  * recursively grep files inside directory: '-r'
